Accordion / Collapsible
#accordion
What is Selenium? #acc-1▾
Selenium is an open-source framework for automating web browsers. It supports multiple languages including Python, Java, C#, and JavaScript. Use find_element or page.locator to interact with elements.
What is Playwright? #acc-2▾
Playwright is a modern automation framework by Microsoft. It supports Chrome, Firefox, and Safari and provides powerful selectors like page.locator('text=...'), page.getByRole(), page.getByTestId() etc.
Which selectors to use? #acc-3▾
Prefer: ID > data-testid > aria-label > CSS selector > XPath. Avoid positional XPath. In Playwright use getByRole for best resilience.
